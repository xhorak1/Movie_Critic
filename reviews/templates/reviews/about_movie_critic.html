About Movie Critic

Welcome to Movie Critic, your go-to platform for discovering and reviewing movies! This project was created to allow users to browse, review, and share their thoughts on various films. Whether you're a movie buff or just looking for something new to watch, Movie Critic has something for everyone.

Features

  • Browse through a comprehensive database of movies.
  • Filter and sort movies by genre, rating, and release date.
  • Read reviews from other users and share your own thoughts.
  • Add new movies to the database (for registered users).

Project Details

This project is built using Django, a high-level Python web framework that encourages rapid development and clean, pragmatic design. The goal is to create a simple, yet powerful platform for movie enthusiasts.

We hope you enjoy using Movie Critic and find it useful for discovering new movies and sharing your opinions!
